# Summary Tables

This folder contains key result tables generated during the analysis of potential drug targets in *Neisseria gonorrhoeae*.

## Table 1: Predicted Cytoplasmic Proteins
This table lists proteins predicted to be localized in the cytoplasm based on PSORTb analysis.  
Cytoplasmic proteins were prioritized as potential therapeutic drug targets.

Columns included:
- Protein ID
- UniProt ID
- Protein name

File:
summary_tables.xlsx (Sheet: Subcellular_Proteins)

## Table 2: Docking and Binding Energy Results
This table summarizes the top candidate compounds identified during the structure-based virtual screening workflow.

Columns included:
- Compound ID
- Docking score
- Glide GScore
- MMGBSA binding energy
- Prime energy

File:
summary_tables.xlsx (Sheet: Docking_Results)

These results highlight the top compounds with strong binding affinity toward the selected drug target.
